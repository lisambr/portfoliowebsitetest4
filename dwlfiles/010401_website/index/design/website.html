Website design

Even though the coding behind this website was a huge part of the project, we had to also strategize our colour scheme, typography, logo and folder structure. You can find more information on all of these elements below.

Colour codes

The different paintbrush strokes show you the main colors of the project. But please note that there are different blue tones to the page in order to make different elements more apparent.

Typography

The typography used on this page is Muli (special shout-out to Google Fonts so I didn't have to go with Arial or something like that). Muli was perfect for the project since it is simple and inspires comfort and peace. It also fits the doodling style of the website perfectly since it will blend in the background and not take over the attention.

For the titles the Lalisa Inc. chose the bold type style.

For the subheaders Lalisa Inc. chose the italic type style.

For the text Lalisa Inc. chose the regular type style.

Logo

The inspiration behind the logo came from my first name Lisa Marie. It is quite simple but has elegance to it which can be adapted to different scenarios.
